## preferences file on windows
C:\Users\USERNAME\AppData\Local\DOSBox

## edit autoexec.bat
see autoexec section in `subl ~/Library/Preferences/DOSBox\ 0.74-3-3\ Preferences`

## mount
```bat
intro mount
mount c ~/c
```

## imgmount
```bat
imgmount b ~/c/disk1.img -t floppy
imgmount -u b   # unmount
```
